# Analyse de Ventes & Dashboard Interactif (Excel / Google Sheets)

## Objectif du Projet
L'objectif de ce projet concret était de nettoyer un jeu de données de ventes d'une boutique en ligne, de calculer des indicateurs clés (KPIs) et de concevoir un tableau de bord interactif pour guider les décisions stratégiques de l'entreprise.

## Compétences & Outils Utilisés
* **Outil :** Google Sheets / Microsoft Excel
* **Data Cleaning :** Identification d'anomalies (quantités négatives, erreurs de texte, formats de date).
* **Formules :** `SOMME`, `NB`/`NBVAL`, calculs de paniers moyens.
* **Analyse :** Tableaux Croisés Dynamiques (TCD) pour regrouper les ventes par catégorie.
* **Visualisation :** Graphiques dynamiques et intégration de segments (Slicers) pour filtrer les données par date/produit.

## Indicateurs Clés Obtenus
* **Chiffre d'Affaires Total :** 525 €
* **Nombre de Ventes :** 7
* **Panier Moyen :** 75 €

## Insights Clés
* La catégorie **Chaussures** est la plus rentable (240 €), suivie de près par les **Vêtements** (225 €).
